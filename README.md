# ColorSkyFun Minecraft Server

欢迎访问我们的Minecraft服务器，你可以加入我们的[QQ频道](https://pd.qq.com/s/40sx6ad1s)来进行交流！



## Our Server

本服务器的`游戏版本`是：`Minecraft Java 1.16.5`，使用`Geysermc`进行基岩版的互通，所以你可以通过`Minecraft Java 1.16.5`或者`我的世界基岩版1.16.100-1.16.221`进行游戏。

- 下载

  1. Android手机

      我们建议你下载 [`我的世界基岩版1.16.220`](https://res.nullatom.com/res/Minecraft/Android/1.16.220.02.apk)

  2. Windows电脑

     我们建议你先安装[JAVA](https://download.oracle.com/java/19/latest/jdk-19_windows-x64_bin.exe)环境,然后下载[`PCL2启动器`](https://ltcat.lanzoum.com/ipFqY0itprkf)安装1.16.5版本（只能1.16.5）

  3. Linux或者Macos

     你需要安装JAVA环境，使用`apt install openjdk-17-jdk`安装，然后下载[`HMCL启动器`](https://hmcl.huangyuhui.net/download/)并使用`java -jar HMCL.jar`命令启动启动器，之后你就可以安装`Minecraft Java 1.16.5`进行游戏了。

  4. 如果你有`Minecarft Java`正版账号，你可以前往[官网](https://www.minecraft.net/zh-hans/download)下载官方启动器

- 加入游戏

  我们的服务器IP：`play.colorsky.fun`

  这对于JAVA版和基岩版是一样的，基岩版的端口为`19132`,这是默认的。
