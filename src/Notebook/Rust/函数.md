# 函数
- 声明方法
    ```rust
    fn main(){
        print!("hi,rust!");
        function_name();
    }

    fn function_name(){
        functiom_body;
    }
    ```
- 函数参数:
    1. 形参(parameters)
    2. 实参(arguments)
    3. 在函数签名里,必须声明每个参数的类型,有多个参数时,每个参数的类型必须单独指明
        ```rust
        fn main(){
            function(5, 6);//实参
        }
        fn function(x: i32, y: u32){//形参
            print!("The x is {}, the y is {}", x, y);
        }
        ```

## 函数体中的语句与表达式
- 语句: 执行一些动作的指令(动作),没有返回值(返回值为`'()'`)
    ```rust
    //一个例子
    fn main(){
        let x = 5 + 6;
    }
    ```
    整个例子就是一个语句,`let x = 5 + 6;`是一个语句,而`5 + 6`则是一个表达式

- 表达式: 计算产生的值(求值),有返回值
